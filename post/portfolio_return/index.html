Calculating index and stock returns using Python and Google Collab

April 3, 2021

Here I want to explore how to calculate index and stock returns on Python.

Info about the data

All the data is gathered using Yahoo API using a library called yfinance.

Google Colab

All the code was run on a google colab, a WEB IDE, and no need for local machine downloads either for the IDE or data.

Now let’s explore the code. Simple and Accurate ; )

Libraries

!pip install yfinance --upgrade --no-cache-dir
import pandas as pd
import yfinance as yf
import matplotlib.pyplot as plt

Importing data

Gathering data from the Yahoo API.

  • Select the tickers
  • Select period
  • Create an empty pandas data frame to collect only adjusted closing prices
tickers = ['HSBA.L','GOOG','^GSPC','^DJI']
start = '2020-01-01'
end = '2021-03-31'
cl_price = pd.DataFrame()

for t in tickers:
  cl_price[t] = yf.download(t,start,end)['Adj Close']

Checking and normalizing the data

  • Run a quick check on the first 5 rows
  • As values have different scales, let’s bring it to the same base
  • Plot normalized values over time
cl_price.head()
(cl_price/cl_price.iloc[0]*100).plot(figsize=(15,6));
plt.show()
  • Index and stock performance over time

Calculating daily and year returns

  • Average trading days per year = 250
index_return = (cl_price / cl_price.shift(1)) - 1
index_return.tail()

annual_index_return = index_return.mean()*250
